# Website Scraping Project

In this project, web scraping techniques are employed to retrieve data from websites. The main tasks included:

- Use BeautifulSoup to parse HTML content from web pages.
- Locate and extract all occurrences of a specific HTML tag in the code.
- Extract and list all hyperlinks present on a webpage.

## Requirements

Python packages used:

- `requests`
- `beautifulsoup4`
